# Progress-list
**Progress List** — это приложение, похожее на «Диспетчер задач», но с дополнительными возможностями.

Оно умеет определять потенциально подозрительные процессы и содержит встроенную базу наиболее распространённых вредоносных процессов.

⚠️ **Важно:** Progress List **не является полноценным антивирусом**. Он помогает обнаружить подозрительную активность, но полностью полагаться на него нельзя. Для проверки и удаления вредоносных программ рекомендуется использовать полноценное антивирусное программное обеспечение.
